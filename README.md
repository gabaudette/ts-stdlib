# ts-stdlib

A collection of TypeScript standard library utilities to enhance productivity and code quality.

## Features

- Utility functions for common tasks
- Type-safe helpers
- Modular and easy to use
- Tree shackable

## Installation

```bash
npm install ts-stdlib
pnpm add ts-stdlib
yarn install ts-stdlib
```
