Generates the Fibonacci sequence up to the specified number of terms.

## Parameters

### n

`number`

The number of terms to generate in the Fibonacci sequence.

## Returns

`number`[]

An array containing the first `n` numbers of the Fibonacci sequence.

## Example

